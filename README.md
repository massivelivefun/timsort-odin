# timsort-odin

A useful generic timsort algorithm implemented in Odin.

## What is Timsort?

Timsort is [this](https://en.wikipedia.org/wiki/Timsort)!

## Usage

Import this package as you would with any other Odin package.
```Odin
import 'timsort:timsort'
```
